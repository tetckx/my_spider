# 京东联盟秒杀专区数据采集器

## 功能特点

- 自动采集京东联盟秒杀专区商品数据
- 文件按日期命名（如：`2026-4-8京东联盟秒杀专区结果.json`）
- 支持导出JSON、CSV和XLSX格式
- 支持修改数据存储路径
- 内置Node.js和Chrome，无依赖直接运行

## 使用方法

### 双击运行

1. 双击 `start.bat`
2. 选择 **1** 开始采集
3. 选择 **2** 修改存储路径
4. 选择 **3** 退出

### 命令行运行

```bash
node scraper.js
```

## 菜单说明

| 选项 | 功能 |
|------|------|
| 1 | 开始采集数据 |
| 2 | 修改存储路径 |
| 3 | 退出 |

## 数据字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 标题 | 商品名称 | 心相印抽纸茶语纸巾...3包体验装 |
| 价格 | 商品到手价 | ￥3.79 |
| 店铺 | 店铺名称 | 绒白旗舰店 |
| 好评数 | 商品好评数量 | 20万+ 好评 |
| 秒杀时间 | 秒杀场次时间 | 2026/4/8 20:00 |

## 环境要求

- Node.js（项目已内置，如无可使用系统Node.js）
- Chrome浏览器（项目已内置，如无可使用系统Chrome）

项目会优先使用系统安装的Chrome和Node.js。

## 项目结构

```
接单测试/
├── start.bat          ← 双击运行
├── scraper.js         ← 爬虫核心
├── config.json        ← 存储路径配置
├── chrome/            ← 内置Chrome
├── node/              ← 内置Node.js
├── node_modules/

```

## 常见问题

### Q: 采集数据为空
A: 可能遇到风控，检查tmp目录下的截图

### Q: 提示找不到Chrome/Node.js
A: 确保系统已安装Chrome和Node.js，或将内置版本放入对应目录

## 技术栈

- Puppeteer-core - 浏览器自动化
- Node.js - 运行环境
- xlsx - Excel导出
